# AMS - Associação dos Moradores de Samambaia
# Sistema para Gestão de Serviços da Associação
A AMS realiza ações, programas e projetos sociais que promovem inclusão, dignidade e o bem estar social.

Esse é um modelo de sistema para gestão da AMS, feito totalmente de forma exemplificativa, para acessar o sistema de gestão aperte "ENTRAR" no login SEM PRECISAR PREENCHER NENHUM CAMPO, (ou seja deixe em branco e aperte fazer login).
